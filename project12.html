Titanic Data Analysis - Survival Insights

Titanic Data Analysis explores survival patterns from the Titanic disaster using exploratory data analysis and visualizations to uncover insights into passenger demographics, ticket classes, and other survival determinants.

Project Overview

This project analyzes the Titanic dataset to derive meaningful insights about survival rates based on factors such as age, gender, class, and family connections. The findings are presented through clear visualizations.

Key Features

  • Exploratory Data Analysis (EDA) on survival data
  • Detailed visualizations for better insights
  • Survival predictions using data patterns

Technologies Used

  • Python
  • Pandas and NumPy for data manipulation
  • Matplotlib and Seaborn for visualizations
